# Vault-First Template

Empty project starter with compact `vault/` as source of truth.

## Use

Copy this folder, rename it, then fill:

- `vault/QUICK_START.md`
- `vault/CODE_MAP.md`
- relevant docs under `vault/wiki/`

Keep `AGENTS.md` in project root so agents start from `vault/QUICK_START.md`.

## Layout

- `AGENTS.md` — agent/project operating rules.
- `vault/` — project memory.
- `src/` — application code placeholder.
- `tests/` — test placeholder.
